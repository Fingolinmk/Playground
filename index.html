Introduction

This project is about a light that is based on an ESP32, programmed with Arduino/PlatformIO. This lamp has three "panels" with 24 LED's, totaling 72 LEDs. All LED's are WS2812 and are therefore individually adressable. This device can be controlled via custom MQTT commands. To control it a simple "Frontend" has been created with Node-Red.
